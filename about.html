Dendrobox is an interactive exploration tool for tree-ring data and its connection to climate.

The data comes from the International Tree-Ring Data Base. Whereever possible, the original chronologies are used, when these were not available, new chronologies have been built from the raw measurements, using spline detrending (50% frequency cutoff at 67% of the curve length, using R package dplR).

In total, 2406 tree-ring chronologies are used for this project. This is not the complete ITRDB data set (roughly 400 series are missing), since I used only the chronologies or raw data files which adhered the ITRDB data formatting standards and could be read in by dplR without errors.

The visualization is done using D3.js and TopoJSON.
